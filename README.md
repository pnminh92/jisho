# Jisho

An english-japanese translator cli

## Installation

```ruby
gem install jisho-cli
```

## Usage

![jisho gif](https://raw.githubusercontent.com/wofi-minh/jisho/master/jisho.gif)

```bash
jisho -w hello

jisho -w "thank you"
```
